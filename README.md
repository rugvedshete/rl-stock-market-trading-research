# rl-stock-market-trading-research
# Reinforcement Learning for Stock Market Trading

Academic research project completed as part of Reinforcement Learning coursework at MIT ADT University.

## Topics
- Markov Decision Processes (MDP)
- Q-Learning
- Deep Q Networks (DQN)
- Proximal Policy Optimization (PPO)
- Reward Engineering
- Portfolio Optimization
- Trading Environment Design

## Outcome
Studied how RL agents can learn adaptive trading strategies through interaction with historical market environments.
